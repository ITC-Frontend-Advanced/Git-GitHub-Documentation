{% tab title="Tabs" %}
### Tabs (Code, Issues, Pull Requests)

* **Code -** The main page where Files live
* **Issues -** track bugs, features, tasks, and discussions
* **Pull Requests -** propose changes, review code, and discuss improvements
* **Actions -** automate workflows such as testing
* **Projects -** manage tasks and progress visually.
* **Others -** Wiki, Security, Insights, Settings ... etc
